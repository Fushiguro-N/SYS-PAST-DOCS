🔄 Diagrama de Secuencia: Registro de Pasantía
Este diagrama ilustra la interacción entre los diferentes componentes del sistema y los actores involucrados (Estudiante, Coordinador y Sistema) durante el proceso de solicitud y aprobación de una pasantía.

Descripción del Flujo
El proceso de registro sigue una secuencia lógica de pasos para garantizar la integridad de la información:

Inicialización: El estudiante solicita una nueva apertura de expediente en el sistema.

Carga Documental: A través de un bucle de iteración, el sistema valida y almacena cada requisito documental enviado por el estudiante.

Selección de Empresa: El sistema consulta la disponibilidad de cupos en la empresa seleccionada antes de realizar la asignación.

Revisión y Aprobación: El Coordinador realiza la revisión formal del expediente. Si el proceso es exitoso, el sistema notifica al estudiante y actualiza el estado del expediente.

Auditoría: Todo el proceso finaliza con un registro de auditoría para garantizar la trazabilidad de la acción.
