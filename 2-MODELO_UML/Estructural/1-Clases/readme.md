# Nivel 3: Diagrama de Clases (Diseño de Componentes)

## Descripción General
El Diagrama de Clases detalla la estructura interna, las entidades de negocio, las propiedades, los métodos principales y las relaciones lógicas del **Sistema de Gestión de Pasantías (SYS PASANTIAS)**. Este diseño organiza el modelo de datos y comportamiento del sistema siguiendo una segmentación analítica por módulos funcionales (Usuarios, Pasantías, Vacantes, Documentación y Auditoría), garantizando la coherencia conceptual del dominio.
